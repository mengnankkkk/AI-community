# 模型效果演示视频拍摄指南

本文档为制作AI虚拟播客工作室的演示视频提供专业建议和脚本模板。

---

## 📹 视频规格建议

### 技术参数
- **分辨率**：1920×1080（1080p）或 1280×720（720p）
- **帧率**：30fps 或 60fps
- **格式**：MP4（H.264编码）
- **时长**：5-8分钟（推荐6分钟）
- **音频**：AAC编码，48kHz，立体声
- **文件大小**：<500MB

### 推荐工具

**录屏软件：**
- **Windows**：OBS Studio（免费）、Camtasia（付费）
- **macOS**：QuickTime Player（自带）、ScreenFlow（付费）
- **Linux**：OBS Studio、SimpleScreenRecorder

**视频剪辑：**
- **免费**：DaVinci Resolve、OpenShot、Shotcut
- **付费**：Adobe Premiere Pro、Final Cut Pro
- **在线**：剪映、必剪

**字幕工具：**
- 剪映（自动识别）
- Arctime（专业字幕）
- Subtitle Edit

### 第二部分：单个播客生成演示（2分钟）

**画面：**
- 完整操作流程录制

**操作步骤演示：**

**步骤1：填写基本信息（20秒）**

**步骤2：配置角色（40秒）**
```

第一个角色，李明，是一位资深的AI研究员。
他的核心观点是：AI将深刻改变人类社会，但需要关注伦理和安全。
我们为他选择"沉稳男声"的音色。

第二个角色，王芳，是一位科技记者。
她关注AI伦理和社会影响。
核心观点是：技术进步需要与人文关怀并重。
为她选择"知性女声"音色。

**步骤3：添加背景知识（可选，20秒）**

系统会自动解析并将相关知识注入到对话中。

```

系统开始工作：
1. 首先，LLM根据角色设定生成对话剧本
2. 然后，TTS引擎将文本转换为语音
3. 最后，进行音频后处理和背景音乐混合

## 🎨 拍摄技巧

### 录屏设置

1. **清理桌面**：关闭无关窗口和通知
2. **调整分辨率**：设置为1920×1080
3. **选择区域**：录制浏览器窗口或全屏
4. **音频源**：使用外置麦克风（推荐）
5. **帧率**：30fps足够，60fps更流畅

### 讲解技巧

1. **语速适中**：不要太快或太慢
2. **停顿得当**：重要内容前后留停顿
3. **重点突出**：关键数字、概念要强调
4. **音量稳定**：保持一致的音量
5. **避免口水音**：多次录制选最佳版本

### 画面处理

**突出重点：**
- 使用圆圈、箭头标注关键区域
- 放大重要细节
- 高亮显示关键文字

**转场效果：**
- 淡入淡出
- 滑动切换
- 不要过度使用特效

**字幕添加：**
- 与讲解同步
- 字体清晰易读（微软雅黑、思源黑体）
- 关键词使用不同颜色
- 字幕位置：底部居中
